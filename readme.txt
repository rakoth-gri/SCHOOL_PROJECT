Для проекта использовать 4 типа шрифта (подключить через google.fonts)

font-family: 'Montserrat', sans-serif; - заголовки h3 и ссылки телефонов в footer
font-family: 'Pacifico', cursive; - Декоративные элементы
font-family: 'Paytone One', sans-serif; - для элементов меню в header
font-family: 'Raleway', sans-serif; - шрифт для текстовых элементов (параграфы)


Иконочный шрифт уже подключен на страницу. Для добавления иконок, используем элемент span:

<span class="icon-circle-right"></span> - стрелка вправо 
<span class="icon-circle-left"></span> - стрелка влево 
<span class="icon-telegram"></span> - иконка телеги
<span class="icon-instagram"></span> - иконка инсты
<span class="icon-plus"></span> - иконка плюса
<span class="icon-facebook"></span> - иконка фейсбук

Вся структура папок уже четко выстроена!
В папке img - имеются фото для каждой секции!

Необходимо написать HTML и СSS-код - для отображения макета в браузере!
